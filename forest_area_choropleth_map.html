Visualisation of the 2019-20 Australian Bushfires

Domain

The domain of the visualisation is natural disasters, specifically bushfires.

Dataset

The dataset is taken from Forests Australia.

The first attribute is state which is of type nominal and represents a grouping.

The second attribute is proportion of total forest area that has been burnt. The attribute type is quantitative as it is numerical and we can perform arithmetic operations with it.

Data Transformations

No data transformations were applied. A normalised field (proportion as %) was already present.

Justification for Idiom Type

I have used a choropleth map to effectively show a numerical attribute for each geographic region (state). The values have importantly been normalised by total forest area for each state. The choropleth map allows us to easily and effectively identify and compare geographic regions and how they have been affected by the 2019-20 bushfire season.
